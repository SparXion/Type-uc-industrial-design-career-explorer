## üìù Typography

### Font Family
- **Primary**: Inter (Google Fonts)
- **Fallback**: system-ui, -apple-system, sans-serif

### Font Sizes
- **2XL**: 2rem (32px) - Page titles
- **XL**: 1.5rem (24px) - Section headers
- **LG**: 1.25rem (20px) - Component titles
- **Base**: 1rem (16px) - Body text
- **SM**: 0.875rem (14px) - Small text
- **XS**: 0.75rem (12px) - Captions

## üìè Spacing System

8px grid system:
- **Space 1**: 0.25rem (4px)
- **Space 2**: 0.5rem (8px)
- **Space 3**: 0.75rem (12px)
- **Space 4**: 1rem (16px)
- **Space 6**: 1.5rem (24px)
- **Space 8**: 2rem (32px)

## üéØ Component Guidelines

### Buttons
- **Primary**: Blue background, white text
- **Secondary**: White background, blue border
- **Size**: Minimum 44px height for accessibility
- **Border Radius**: 8px

### Inputs
- **Border**: 2px solid
- **Border Radius**: 8px
- **Focus**: Blue border with shadow
- **Padding**: 12px 16px

### Cards
- **Background**: White
- **Border**: 1px solid light gray
- **Border Radius**: 12px
- **Shadow**: Subtle drop shadow
- **Padding**: 24px

## üé¨ Animations

### Transitions
- **Duration**: 0.3s
- **Easing**: ease-out
- **Hover**: Scale 1.05
- **Focus**: Scale 1.02

### Page Transitions
- **Fade In**: 0.6s ease-out
- **Slide Up**: 0.4s ease-out

## üì± Responsive Design

### Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

### Mobile-First Approach
- Start with mobile design
- Progressive enhancement
- Touch-friendly interactions
- Readable text sizes

## ‚ôø Accessibility

### Color Contrast
- **AA Standard**: 4.5:1 minimum
- **AAA Standard**: 7:1 for important text

### Focus Indicators
- **Visible**: 2px blue outline
- **Keyboard Navigation**: Tab order
- **Screen Readers**: Proper ARIA labels

## üé® Visual Hierarchy

### Information Architecture
1. **Primary**: Main content
2. **Secondary**: Supporting content
3. **Tertiary**: Additional details

### Visual Weight
- **Heavy**: Bold text, large sizes
- **Medium**: Regular text, medium sizes
- **Light**: Small text, muted colors
